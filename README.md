# Overview
Breachcord is a Discord bot made with discord.py that helps with data breach analysis and digital forensics. 
<br><br> Since itâ€™s self-hosted right now, you can run it by forking the repo and running `start.py`.

# Installation

[Installation](#howtoinstall) is **simple**, you can currently install Breachcord by:

- Local Installation (by forking the repository)
  <br>
  1.) Fork this repository by following the steps in the [official GitHub guide](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/fork-a-repo) or you can find the [installation guide here](https://github.com/z3rodaycve/breachcord/wiki/Installation-Guide).<br>
  2.) After forking, clone your fork locally and run `start.py` with **Python 3.10+**
  
If you need help with installing Breachcord, feel free to contact me anytime using any of the details listed on my [GitHub profile](https://github.com/z3rodaycve).
